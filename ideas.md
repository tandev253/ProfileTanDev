# Brainstorm Thiết Kế Portfolio Lập Trình Viên

## Ý Tưởng 1: Modern Minimalist (Xác Suất: 0.08)

**Design Movement**: Swiss Design + Digital Modernism
- Tối giản nhưng mạnh mẽ, tập trung vào nội dung
- Lưới chặt chẽ, căn chỉnh hoàn hảo
- Không có trang trí thừa

**Core Principles**:
1. Clarity over decoration - Mọi phần tử phục vụ mục đích
2. Typography as design - Font lớn, bold, tạo visual hierarchy
3. Negative space - Khoảng trắng rộng rãi tạo cảm giác thoải mái
4. Monochromatic + accent - Đen/trắng + một màu accent mạnh

**Color Philosophy**:
- Nền: Trắng sạch (hoặc off-white)
- Text: Đen sâu hoặc charcoal
- Accent: Một màu neon (xanh lá cây neon, cam, hoặc tím)
- Lý do: Tạo sự tập trung, chuyên nghiệp, hiện đại

**Layout Paradigm**:
- Hero section: Tên lớn + một hình ảnh accent
- Sections xếp chồng lên nhau, mỗi section có breathing room
- Sidebar hoặc sticky nav cho navigation
- Cards cho skills/projects nhưng không quá nhiều

**Signature Elements**:
1. Geometric shapes (circles, lines) làm dividers
2. Bold typography headers
3. Subtle hover effects (scale, color change)

**Interaction Philosophy**:
- Smooth scroll transitions
- Hover states rõ ràng
- Click feedback tức thì

**Animation**:
- Fade-in on scroll
- Subtle scale on hover
- Smooth color transitions (0.3s)

**Typography System**:
- Display: Poppins Bold 48-64px
- Heading: Poppins SemiBold 28-36px
- Body: Inter Regular 16px
- Accent text: Poppins Medium

## Ý Tưởng 2: Tech-Forward Dark (Xác Suất: 0.07)

**Design Movement**: Cyberpunk + Developer Culture
- Tối, hiện đại, cảm giác "hacker aesthetic"
- Neon colors trên nền tối
- Code-inspired elements

**Core Principles**:
1. Dark first - Nền tối bảo vệ mắt
2. Neon accents - Màu sáng tạo contrast mạnh
3. Grid-based - Lưới 8px, cấu trúc rõ ràng
4. Tech elements - Icons, code snippets, terminal vibes

**Color Philosophy**:
- Nền: Charcoal/dark navy (#0f0f0f hoặc #1a1a2e)
- Text: Xanh lá cây neon (#00ff00) hoặc xanh cyan (#00ffff)
- Secondary: Tím neon (#ff00ff)
- Lý do: Tạo cảm giác tech-savvy, hấp dẫn developer

**Layout Paradigm**:
- Asymmetric layout - Một bên text, một bên visual
- Floating cards với blur background
- Vertical timeline cho projects
- Code block showcase

**Signature Elements**:
1. Neon glowing effects
2. Terminal-style text
3. Grid background pattern

**Interaction Philosophy**:
- Glow effects on hover
- Typing animations
- Matrix-like transitions

**Animation**:
- Glow pulse effect
- Text typing animation
- Smooth fade with glow

**Typography System**:
- Display: Space Mono Bold (monospace)
- Heading: Courier New hoặc JetBrains Mono
- Body: Roboto Mono Regular
- Accent: Space Mono Bold

## Ý Tưởng 3: Creative Gradient (Xác Suất: 0.09)

**Design Movement**: Contemporary Design + Playful Tech
- Gradients mềm mại, không aggressive
- Colorful nhưng balanced
- Modern + accessible

**Core Principles**:
1. Gradient backgrounds - Mềm mại, không quá sáng
2. Rounded corners - Friendly, approachable
3. Depth layering - Shadows + overlays tạo depth
4. Emoji + icons - Playful touches

**Color Philosophy**:
- Base gradient: Từ xanh dương (#3b82f6) sang tím (#8b5cf6)
- Accent: Cam (#f97316) hoặc hồng (#ec4899)
- Nền cards: Trắng semi-transparent hoặc blur
- Lý do: Tạo cảm giác modern, friendly, creative

**Layout Paradigm**:
- Hero section lớn với gradient background
- Floating cards trên gradient
- Curved dividers giữa sections
- Blobby shapes làm accent

**Signature Elements**:
1. Soft gradients
2. Rounded blob shapes
3. Layered depth effects

**Interaction Philosophy**:
- Smooth transitions
- Cards lift on hover
- Gradient shifts on interaction

**Animation**:
- Floating animation cho cards
- Gradient shift on scroll
- Smooth bounce on click

**Typography System**:
- Display: Outfit Bold 48-56px
- Heading: Outfit SemiBold 28-32px
- Body: Figtree Regular 16px
- Accent: Outfit Medium

# Lựa Chọn Thiết Kế

**Chọn: Modern Minimalist**

Lý do:
- Phù hợp nhất cho portfolio lập trình viên (chuyên nghiệp, rõ ràng)
- Dễ đọc, tập trung vào nội dung (kỹ năng, dự án)
- Có thể thêm accent color để tạo visual interest
- Tính scalability cao - dễ mở rộng khi thêm projects

Màu sắc cuối cùng:
- Nền: Trắng (#ffffff)
- Text chính: Charcoal (#1a1a1a)
- Text phụ: Gray (#666666)
- Accent: Xanh lá cây neon (#10b981 - emerald)
- Border/divider: Light gray (#e5e7eb)

Typography:
- Display: Poppins Bold
- Heading: Poppins SemiBold
- Body: Inter Regular
- Code: JetBrains Mono

Layout:
- Hero section: Tên lớn + brief description
- About: Short bio
- Skills: Grid layout cho technologies
- Projects: Card layout
- Contact: Footer với links
